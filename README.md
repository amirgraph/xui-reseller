## ✨ نهان چیست؟

**نهان** یک اکوسیستمِ آمادهٔ فروشِ VPN است که برای شرایطِ سختِ فیلترینگ طراحی شده.
به‌جای اینکه ساعت‌ها x-ui و WARP و nginx و ربات را دستی سرهم کنی، **یک دستور می‌زنی، چند سوال جواب می‌دهی، و کلِ کسب‌وکارت بالا می‌آید** — با پنلِ نماینده، رباتِ تلگرام، قیمت‌گذاری، و ضدفیلترِ چندلایه.

> ساخته‌شده روی درس‌های واقعیِ میدان: پایداری زیر بار، دورزدنِ فیلترینگِ دامنه و IP، و تجربهٔ روانِ مشتری.

## 🚀 نصب — یک دستور

```bash
git clone https://github.com/<user>/nahan.git
cd nahan
sudo bash setup.sh
```

نصب‌کننده این‌ها را می‌پرسد و بقیه را خودش انجام می‌دهد:

```
۱/۶  دامنه‌ها ............ دامنهٔ اصلی + دامنهٔ Cloudflare
۲/۶  تلگرام و ادمین ...... توکن ربات، آیدی ادمین، رمز پنل
۳/۶  x-ui ................ یوزر/رمز (مسیر و کلید API خودکار)
۴/۶  قیمت‌گذاری .......... قیمت پنل، هر گیگ، نامحدود، شارژ
۵/۶  سرور ................ IP (خودکار تشخیص)
۶/۶  کلیدهای امنیتی ...... JWT و ... خودکار ساخته می‌شوند
```

## 🧭 معماری

```
                کاربر (ایران)
      ساب اصلی → Cloudflare (ساب‌دامینِ تصادفیِ نهان)
      ساب پشتیبان → CDN داخلی (دامنهٔ اصلی)
                      │
                      ▼
        ┌─────────────────────────────┐
        │   nginx :443  (TLS + مسیریابی) │
        └───────────────┬─────────────┘
             ┌──────────┴───────────┐
             ▼ تونل نهان            ▼ پنل و ساب
        xray inbound :8001      node app :3000
        (بلاک تبلیغات + تورنت)   (پنل + ربات + دیتابیس)
             │
             ▼
        WARP (wireproxy — خروجیِ مخفی)
```

## 📦 چه چیزی نصب می‌شود؟

| ماژول | نقش |
|-------|-----|
| 🧩 **x-ui** | هستهٔ مدیریتِ کاربران و xray (MHSanaei/3x-ui v3) |
| 🕳️ **نهان** | اینباندِ xhttp پشتِ CDN + قالبِ routing با بلاکِ تبلیغات |
| 🌐 **WARP** | خروجیِ userspace با wireproxy (پایدار) |
| 🔀 **nginx** | پایانهٔ TLS روی ۴۴۳، مسیریابیِ تونل و پنل |
| 🔎 **اسکنر IP تمیز** | یافتن و چرخشِ خودکارِ IPهای تمیزِ کلادفلر |
| 🖥️ **پنل + ربات** | پنلِ رزلر (PWA) + رباتِ تلگرام + قیمت‌گذاری |
| 🩺 **تیونینگ‌ها** | conntrack، watchdog حافظه، جعبه‌سیاه، محدودیت لاگ |

---

## ⚙️ بعد از نصب (چند قدم دستی)

1. **Cloudflare** — ۳ ساب‌دامینِ نهان را (که نصب‌کننده نشان می‌دهد) با پروکسیِ نارنجی به IP سرور بزن.
2. **CDN داخلی (Arvan/…)** — دامنهٔ اصلی را به سرور وصل کن و **کش را برای `/sub` روشن کن** (تا از داخل ایران بدون فیلترشکن باز شود).
3. **رباتِ تلگرام** — توکن را از [@BotFather](https://t.me/BotFather) بگیر و هنگام نصب وارد کن.

---

## 🔐 امنیت

- **صفر secret در ریپو** — این ریپو با خیال راحت public می‌شود.
- رمزها با ورودیِ مخفی پرسیده می‌شوند؛ `JWT` و کلیدها خودکار ساخته می‌شوند.
- `.env`، دیتابیس‌ها، پروفایلِ WARP و کلیدها همه در `.gitignore`.
