# نهان — نصب‌کنندهٔ خودکارِ اکوسیستمِ VPN رزلر

نصبِ کاملِ یک سیستمِ فروشِ VPN ضدفیلتر با **یک دستور** و پرسش‌های تعاملی.
همه‌چیز خودکار: `x-ui` + `xray` (تونل نهان xhttp) + `WARP` + `nginx` + اسکنر IP تمیزِ Cloudflare + پنل و رباتِ رزلر.

## معماری

```
کاربر (ایران)
   │  ساب اصلی → Cloudflare (ساب‌دامین‌های نهان) ─┐
   │  ساب پشتیبان → CDN داخلی (دامنهٔ اصلی) ──────┤
   ▼                                              ▼
        nginx :443  (stream + ssl_preread، مسیریابی بر اساس SNI)
                    │
      ┌─────────────┴───────────────┐
      ▼ نهان (xhttp)                ▼ پنل/ساب
   xray inbound :8001            node app :3000
      │ (routing: بلاک تبلیغات+تورنت)
      ▼
   WARP (wireproxy، خروجی مخفی)
```

## نصب

```bash
git clone <repo>
cd nahan-installer
sudo bash setup.sh
```

نصب‌کننده می‌پرسد: دامنه‌ها، توکن ربات، آیدی ادمین، رمزها، **قیمت‌گذاری** (قیمت پنل، هر گیگ، نامحدود، شارژ کارت/کریپتو)، و بقیه.
هیچ مقدارِ حساسی در ریپو نیست — همه هنگام نصب پرسیده یا تولید می‌شود.

## اجزا (ماژول‌ها در `scripts/`)

| ماژول | کار |
|-------|-----|
| `00-deps.sh` | نصب پیش‌نیازها (node, nginx, ابزارها) |
| `10-warp.sh` | ثبتِ WARP تازه (wgcf) + wireproxy + systemd |
| `20-xui.sh` | نصب x-ui + اینباند نهان + قالب routing (بلاک تبلیغات) + کلید Reality |
| `30-app.sh` | استقرار پنل/رباتِ node + دیتابیس + ادمین + قیمت‌ها + pm2 |
| `40-nginx.sh` | کانفیگ nginx (stream SNI + http) با دامنه‌های تو |
| `50-scanner.sh` | اسکنر IP تمیز + provision + cronها |
| `60-tunings.sh` | تیونینگ پایداری (conntrack, memwatch, journald, swappiness) |
| `99-verify.sh` | بررسی سلامتِ نهایی |

## بعد از نصب (کارهای دستی)

1. در **Cloudflare**: ۳ رکورد A برای ساب‌دامین‌های نهان (نمایش داده می‌شوند) با پروکسیِ نارنجی به IP سرور.
2. در **CDN داخلی (Arvan/…)**: دامنهٔ اصلی را به IP سرور وصل کن + **کش را برای مسیر `/sub` روشن کن** (تا از داخل ایران بدون فیلترشکن باز شود).
3. توکن رباتِ تلگرام را از BotFather بگیر و هنگام نصب وارد کن.

## امنیت

- `.env`، دیتابیس‌ها، فایل‌های WARP و کلیدها در `.gitignore` — هیچ‌وقت commit نمی‌شوند.
- رمزها با `read -s` پرسیده می‌شوند (نمایش داده نمی‌شوند).
- `JWT_SECRET`، کلید API x-ui، و کلیدهای Reality **خودکار تولید** می‌شوند.
